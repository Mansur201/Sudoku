Метод ограничения доменов (Constraint Propagation) в контексте решения судоку заключается в том, чтобы для каждой клетки отслеживать набор возможных значений, которые можно вставить в неё. По мере заполнения клеток уменьшаются возможности для соседних клеток, и мы постепенно сужаем количество допустимых кандидатов для остальных клеток.

Основная идея:
Для каждой пустой клетки храним набор допустимых значений (кандидатов), исходя из текущего состояния головоломки.
Когда заполняется клетка, убираем это значение из наборов возможных кандидатов для всех клеток в той же строке, столбце и блоке.
Если для какой-то клетки остаётся только одно возможное значение, заполняем её.
